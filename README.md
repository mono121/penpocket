# penpocket

所持・気になっている文房具のメモ用サイトです。

## 概要

`penpocket` は、文房具メモ用の静的ウェブサイトです。
- `collection.html` で所有文房具をジャンル別に表示
- `brands.html` でお気に入りブランドの紹介
- `wishlist.html` で欲しい文房具をリストアップ

## ファイル構成

- `index.html` - トップページ
- `collection.html` - 所有文房具のコレクション
- `brands.html` - 文房具ブランド紹介
- `wishlist.html` - 欲しい文房具リスト
